# Milestone 1 – Zakończenie etapu koncepcyjnego

**Data zakończenia:** 24.03.2026  
**Status:** Zrealizowany  

## Cel etapu
Celem etapu było opracowanie kompletnej koncepcji gry oraz przygotowanie dokumentacji projektowej stanowiącej podstawę dalszych prac produkcyjnych.

## Zakres prac
- Opracowanie i zatwierdzenie dokumentu Game Design Document (GDD)
- Zdefiniowanie głównej pętli rozgrywki (cykl dzień/noc)
- Określenie kluczowych mechanik gry:
  - system ekonomii
  - system walki
  - elementy survivalowe
- Zaprojektowanie trybu multiplayer (kooperacja, skalowanie trudności)
- Określenie struktury świata gry oraz głównych lokacji
- Podział ról i obowiązków w zespole projektowym
- Wybór technologii oraz narzędzi produkcyjnych

## Rezultaty
- Kompletny i zatwierdzony dokument GDD
- Zdefiniowane systemy i mechaniki gry
- Przygotowana koncepcja architektury projektu

## Kryteria ukończenia
- Dokument GDD został zatwierdzony przez wszystkich członków zespołu
- Zakres funkcjonalny projektu został jasno określony
- Każdy członek zespołu posiada przypisaną rolę

# Milestone 2 – Setup projektu i przygotowanie środowiska

**Data zakończenia:** 28.03.2026  
**Status:** Zrealizowany  

## Cel etapu
Przygotowanie środowiska developerskiego umożliwiającego rozpoczęcie implementacji oraz efektywną współpracę zespołu.

## Zakres prac
- Konfiguracja silnika gry
- Utworzenie projektu oraz repozytorium kodu źródłowego
- Opracowanie struktury katalogów projektu
- Konfiguracja systemu kontroli wersji
- Import zasobów tymczasowych (placeholderów graficznych)
- Przygotowanie podstawowej sceny startowej

## Rezultaty
- W pełni skonfigurowane środowisko produkcyjne
- Repozytorium umożliwiające pracę zespołową
- Gotowa struktura projektu do dalszej implementacji

## Kryteria ukończenia
- Projekt uruchamia się poprawnie na stanowiskach wszystkich członków zespołu
- Repozytorium działa stabilnie i umożliwia współdzielenie kodu
- Możliwe jest rozpoczęcie prac nad mechanikami gry
