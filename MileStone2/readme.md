# Milestone 2 – Prototyp funkcjonalny (Core Gameplay)

**Data zakończenia:** 14.04.2026  
**Status:** Zrealizowany  

## Cel etapu
Stworzenie grywalnego prototypu obejmującego podstawowe mechaniki oraz główną pętlę rozgrywki.

## Zakres prac

### System gracza
- Implementacja sterowania postacią
- Obsługa kolizji i poruszania się po mapie
- Implementacja kamery śledzącej gracza

### Środowisko gry
- Utworzenie podstawowej mapy (blok oraz otoczenie)
- Przygotowanie przestrzeni do rozgrywki

### Główna pętla rozgrywki
- Implementacja cyklu dzień/noc
- Logika przełączania faz gry (eksploracja / walka)

### System przeciwników
- Implementacja podstawowego AI przeciwników
- System pojawiania się przeciwników w nocy
- Podstawowe zachowania (ruch w kierunku gracza, atak)

### System walki
- Implementacja podstawowego systemu walki
- Zadawanie i odbieranie obrażeń
- Prosty system zdrowia (HP)

## Rezultaty
- Grywalny prototyp zawierający podstawową pętlę rozgrywki
- Możliwość testowania interakcji między systemami

## Kryteria ukończenia
- Gracz może swobodnie poruszać się po mapie
- System dzień/noc działa poprawnie
- Przeciwnicy pojawiają się i reagują na gracza
- Możliwa jest podstawowa walka
- Prototyp działa stabilnie (brak błędów krytycznych)
